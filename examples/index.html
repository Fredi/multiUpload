Photo Album

This is a simple way to create photo albums using multiUpload. The basic idea is: You have a form that will be sent using Ajax to save the record and return the id inserted. Then a multiUpload instance is created to upload all images along with the album's id.


Photo Album 2

This is an improved Photo Album that creates thumbs and allows you to save a caption for each uploaded image.
